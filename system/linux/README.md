## 使用 Git Bash 命令行

可以使用 `Git Bash` 程序直接连接登陆服务器, 简单快捷而且有效

连接命令:

```shell
ssh <login_user_name>@<server_address>
```

比如:

```shell
ssh ubuntu@58.87.71.34
```
