# Nginx Web反向代理服务器

## 使用

### 关闭 Windows 版

查看nginx的进程占用情况
```cmd
tasklist | find /i "nginx.exe" || exit
```

关闭nginx的所有进程
```cmd
taskkill /im nginx.exe /f
```
