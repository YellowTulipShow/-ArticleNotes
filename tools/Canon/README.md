* [学习链接](https://ugp01.c-ij.com/ij/webmanual/Manual/All/MG3000%20series/SC/CNT/Top.html)

## 安装驱动步骤:

    访问网址: http://canon.com/ijsetup
    点击网站中显示的 "设置"
    输入型号 "MG3080" 点击 "前往"
    下载文件驱动文件
    打开驱动, 点击 "开始设置"
