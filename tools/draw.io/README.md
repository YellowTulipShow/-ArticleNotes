# Draw.io

非常好用且免费开源的流程图控制软件

且可以生成xml文本文件用于版本控制

支持离线编辑, 粘贴截图, 自定义绘制图案, 非常强大!!!

支持数学公式: 将原 **\$xxx\$** 格式 改为 **\`xxx\`** 使用
